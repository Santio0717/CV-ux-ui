Prototipado

El prototipo se desarrolló para simular de forma clara el flujo completo del sistema, integrando navegación, estados visuales y pantallas clave del recorrido del usuario.

  • Modelado 3D de los personajes en Blender.
  • Conexión del flujo completo: menú, calendario, perfil y edición.
  • Interacciones fluidas entre pantallas.
  • Jerarquía visual para facilitar lectura y navegación.
  • Prototipo funcional listo para pruebas de usabilidad.
