Accesibilidad

La accesibilidad fue un aspecto transversal durante todo el desarrollo del proyecto. Se aplicaron principios orientados a mejorar la experiencia de usuarios con diferentes capacidades visuales y niveles de interacción, garantizando una navegación más inclusiva.

Se priorizó el uso de contrastes adecuados, tipografía legible, jerarquías claras, tamaños consistentes y componentes fáciles de identificar. Asimismo, se planteó una estructura compatible con buenas prácticas de accesibilidad para facilitar el uso mediante lectores de pantalla y navegación mediante teclado.

Estas decisiones permitieron construir una experiencia más comprensible y eficiente sin afectar la identidad visual del proyecto.

Resultado: una interfaz diseñada bajo principios de accesibilidad que mejora la legibilidad, la comprensión y la inclusión de distintos tipos de usuarios.
